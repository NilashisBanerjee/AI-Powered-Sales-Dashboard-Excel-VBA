# AI-Powered Sales Performance Dashboard

## Overview

This project showcases a fully automated Sales Performance Dashboard built using Microsoft Excel, VBA, Pivot Tables, Interactive Charts, and AI-assisted code generation.

The dashboard automatically transforms raw sales data into meaningful business insights, helping users track sales trends, profitability, regional performance, category-wise sales, and discount impact.

## Features

✅ Automated Data Cleaning

✅ Automated Pivot Table Creation

✅ Interactive KPI Cards

* Total Sales
* Total Profit
* Total Orders
* Total Quantity Sold
* Average Discount
* Profit Margin

✅ Dynamic Visualizations

* Sales by Segment
* Sales & Profit Trend
* Monthly Sales Performance
* Profit vs Discount
* Sales by Category
* Sales by Region

✅ Interactive Slicers

* Year Filter
* Month Filter

✅ VBA Automation

* One-click Dashboard Generation
* Automatic Chart Formatting
* Dynamic KPI Updates
